# E-Commerce Cart Management System

This is an E-Commerce Cart Management System built to manage products, shopping cart, and checkout process for an online store.

## Features

- **Product Management**: Add, update, and remove products from the store.
- **Shopping Cart**: Users can add/remove products to/from the cart.
- **Checkout**: Users can proceed to checkout and view cart summary.
- **User Authentication**: Admin and user roles for product management.
